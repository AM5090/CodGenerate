# CodGenerate - OTP-сервис с поддержкой нескольких каналов доставки

Сервис для генерации и валидации OTP-кодов (одноразовых паролей) с доставкой через SMS (SMPP эмулятор), Email, Telegram и сохранением в файл.

## 🛠 Технологии

| Компонент | Технология |
|-----------|-----------|
| Язык | Java 21 |
| Фреймворк | Spring Boot 4.0.6 |
| База данных | PostgreSQL 17 (Docker) |
| Аутентификация | JWT (Access + Refresh токены) |
| ORM | Hibernate / JPA |
| Миграции | Flyway |
| Почта | Jakarta Mail (Angus Mail) + SMTP |
| SMS | JSMPP + SMPPsim (эмулятор) |
| Telegram | Telegram Bot API |
| Сборка | Maven |
| Логирование | SLF4J + Logback |

## 🚀 Запуск приложения

### 1. Предварительные требования

- Docker и Docker Compose
- Java 21 (JDK)
- Maven

### 2. Запуск PostgreSQL в Docker

```bash
    docker-compose up -d
```

## Если таблицы БД не создались через Flyway
Подключитесь к БД (localhost:5433, БД: cg_app, пользователь: postgres, пароль: postgres123) и выполните SQL-скрипт из src/main/resources/db/migration/V1__create_tables.sql.

### Сборка и запуск
mvn clean compile
mvn spring-boot:run

## 📡 API эндпоинты

### Публичные

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/api/users/register` | Регистрация нового пользователя |
| POST | `/api/users/login` | Вход, получение JWT токенов |
| POST | `/api/auth/refresh` | Обновление Access токена |

### Пользовательские (требуют JWT)

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/api/otp/generate` | Генерация OTP кода |
| POST | `/api/otp/validate` | Проверка OTP кода |
| GET | `/api/otp/status/{operationId}` | Статус OTP |

### Административные (только роль ADMIN)

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/admin/users` | Список всех пользователей (кроме админов) |
| DELETE | `/api/admin/users/{login}` | Удаление пользователя |
| GET | `/api/admin/otp/config` | Текущая конфигурация OTP |
| PUT | `/api/admin/otp/config` | Изменение конфигурации OTP |
